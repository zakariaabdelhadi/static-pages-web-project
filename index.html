Description

Le livre "Introduction à la programmation" est une tentative d'introduire la programmation aux débutants. Pourquoi apprendre à programmer ? Pour répondre, voici une citation par Rick Cook dans son livre


Rick Cook
The Wizardry Compiled

Aujourd'hui, la programmation est devenue une course entre le développeur, qui s'efforce de produire de meilleures applications à l'épreuve des imbéciles et l'univers, qui s'efforce de produire de meilleurs imbéciles. Pour l'instant, l'univers a une bonne longueur d'avance


Contenu

Dans le chapitre 1, on présentra des notions nécessaires pour comprendre la programmation tel que le bit. Quelques représentations des nombres et du texte seront entamées. En plus, les technologies de langages et d'exécusion seront présentées.


Plus de concepts seront ajoutées dans des versions ultérieurs.

Le chapitre 2 est réservé pour les concepts de base de la programmaion. Les commentaires, les variables et les instructions sont parmi les concepts présentés dans ce chapitre.
